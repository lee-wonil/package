[CSS]

1. CSS (Cascading Style Sheets)
	HTML과 함께 웹 표준의 기본 개념
	HTML : 웹사이트의 내용 나열
	CSS  : 디자인 구성

	독립적으로 사용 불가. HTML에 도움을 주는 언어.
	HTML에서 불가능한 디테일한 디자인 기능

2. 스타일 형식

	선택자{ 속성명: 속성값; 속성명: 속성값; ........}
	
	선택자 	: selector 라 하며, 스타일을 어디에 적용할 것인지 지정하는 것.(*)
	{}	: 스타일 적용 코드 영역
	:	: 속성명과 속성값의 구분자
	;	: 속성명:속성값 뒤에 명령어의 종료를 나타내기 위해 기입

	/* */	: css 내에서 주석 표현 방법

3. CSS 적용 방법 3가지
	: 스타일 규칙들을 한데 묶어 놓은 것을 "스타일 시트"stylesheet라 함.

	1) 외부 스타일 시트	: CSS 파일 별도로 작업하여 연결 (가장많이 사용)

		<link href ="css1.css" rel="stylesheet" type="text/css"/>
		href : css 파일 경로 지정
		rel : 링크걸린 파일과의 관계에 대한 기술
		type : 파일의 형식 기술 : text로 기술된 css 타입이다.
	2) 내부 스타일 시트	: HTML파일에 <style> 태그로 작업
	3) 인라인 스타일 시트	: 태그에 속성으로 작업
	
	
4. 선택자 (selector)(*)	
	
	1) 전체 선택자
		* : 모든 요소에 스타일 적용
		
		* { 속성:속성값;....}

	2) 태그 선택자
		태그명 : 해당 태그명에 모두 적용
		
		h1 { 속성 : 속성값;.... }
		--> 문서 내의 모든 h1태그에 적용
	3) class 선택자
		.class 값 : 특정 부분에 스타일 적용. 그룹 지정하여 적용 하듯이 사용 가능.
		
		.name { 속성: 속성값;.... }

	4) id 선택자
		#id값 : 특정 부분에 스타일 적용. id 값은 문서내에서 유일해야 하므로
			유일한 특정 한개의 요소에만 적용할 때 사용

		#container { 속성 : 속성값;....} 
		
	5) 여러 선택자 동시 적용
		: 선택자를 쉼표를 구분자로 나열하여 적용

		선택자, 선택자{}
	6) 스타일 적용 우선순위
		#1. 인라인스타일 > id 스타일 > class 스타일 > 태그 스타일
		#2. 코드 순서 : 나중에 기술한것이 적용

		* 스타일 상속 : 부모 태그에 적용된것은 자식태그에도 적용됨
			ex ) body에 폰트 적용하면 그 안에 있는 태그들도 모두 적용됨.
		
	7) 연결 선택자
		#1) 하위 선택자
			: 띄어쓰기를 구분으로 부모 요소에 포함된 하위 선택 요소에 모두 적용
			
			상위선택자 하위선택자{....}
		#2) 자식 선택자
			: > 를 구분자로 하위 요소 전체가 아니라 자식 요소에만 스타일 적용. 손자 적용X
			부모선택자 > 자식선택자 {...}
		#3) 인접 형제 선택자
			: 가장 가가운 형제 요소에 스타일 적용
			  앞 선택자 바로 다음에 오는 선택자가 지정
			선택자(형)+ 선택자(아래동생){...}
		#4) 형제 선택자
			: 인접과 달리 모든 형제 요소에 적용. 형 다음에 붙어 있는 것만 적용.

			선택자(형) ~ 선택자(동생들) {...}
		
	8) 속성 선택자
		#1) 속성	: 지정한 속성을 가진 요소를 찾아 적용
			선택자[속성]{...}
		#2) 속성값 : 속성에서 나이가 속성의 값이 일치하는 요소를 찾아 스타일 적용
			선택자[속성=값]{...}
		#3) 속성 ^=값 : 특정 값으로 시작하는 속성에 스타일 적용
			선택자[속성^=값]{...}
		#4) 속성 $=값 : 특정 값으로 끝나는 속성에 스타일 적용
			선택자[속성$=값]{...}
		#5) 속성 *=값 : 특정 값의 일부가 일치하는 속성에 스타일 적용
			선택자[속성*=값]{...}
	
	9) 가상 클래스
		: 사용자 동작에 반응하는 스타일 적용은 가상 클래스 선택자를 사용.
		  <a> 태그에서 많이 볼 수 있다.

		a:link		: 기본. 방문전 링크 스타일 적용
		a:visited	: 방문한 링크 스타일 적용
		a:hover		: 요소에 마우스 커서를 올려놓았을때 스타일 적용(mouseover/rollover)
		a:active	: 요소 활성화 했을때(누르고 있을 때) 스타일 적용
		a:focus		: 요소에 초점이 맞추어 졌을 때 스타일 적용
		
		스타일 적용시 순서 유의 : link, visited, hover, active 순서대로 적용해야 제대로 적용됨
		
	10) UI요소 상태에 따른 가상 클래스
		: 웹 소의 상태에 따라 스타일 지정
		
		:enable		: 텍스트 필드
		:disable
		:checked	: 라디오, 체크박스

	11) 구조 가상 클래스
		: 웹 문서 구조를 기준으로 특정 위치에 있는 요소 찾아 스타일 적용 
	
		:root			: HTML
		:nth-child(n)		: 앞에서부터 n번째 자식 요소		
		:nth-last-child		: 끝에서부터 n번째 자식 요소	
		:nth-child(odd)		: 홀수 번째 자식요소
		:nth-child(even)	: 짝수 번째 자식요소

		first-child		: 첫번째 자식 요소
		last-child		: 마지막 자식 요소

		not()			: 괄호안에 있는 요소를 제외한...

	12) 가상 요소
		: 가상클래스 = 여러 태그 중 원하는 태그를 선택하기 위한 선택자
		  가상 요소 = 내용의 일부만 선택해 스타일 적용
		  * 구별을 위해 앞에 콜론 두개(::)붙여 표시

		
		::first-line
       		::first-letter
       		::before
        	::after

6. 박스 모델
	1) 블록 레벨	: 혼자 한줄 차지하는 요소. 박스 형태	블록 태크 : p,hn, ul, ol, div,form,table 등
	   인라인 레벨	: 혼자 한줄 차지하지않는 요소		인라인 : img, br, span, input, textarea, button 등

	2) display : 화면 배치 방법 결정
		none	: 화면에 표시하지 않음
		block	: 블록레벨로 지정
		inline	: 인라인 레벨로 지정. 한줄로 배치가능하나 너비나 높이, 마진같은 박스모델 값이 정확히 적용되지 않음
		inline-block	: 인라인 레벨요소이면서 블럭레벨 속성을 갖도록 지정한 후 마진 값을 준것

	3) border : 테두리
		border-width, border-color 등 속성이 분리 적용가능하게 있지만 간단하게는
		border : 두께 스타일 색상
		그 외에 border-top / left /right / bottom, border-top-left-radius 같이 모서리 둥글게도 가능
	
	4) 여백
		margin 속성	: 요소 바깥 여백. 값1개 : 상하좌우, 값2개 : 상하, 좌우
						  값 4개 : 상, 우, 하, 좌(시계방향)
		padding 속성	: 컨텐츠와 테두리 사이 여백 margin과 값 적용 방법 동일.

7. 포지셔닝
	1) box-sizing	: 박스 너비 ㅈ정하기
		width, height, margin, padding, border 지정
		
		박스모델의 너비  = 컨텐츠크기 + 좌우 padding + 좌우 border 두께(+ 좌우 margin = 전체요소 너비)

		box-sizing 속성값
			content-box : width 속성값을 컨텐츠 영역의 너비값으로 사용. 기본값
			border-box  : width 속성값을	

	2) float 속성
		: 요소를 문서위에 떠있게 만들어 왼쪽/오른쪽 으로 배치

		left	: 왼쪽	배치
		right	: 오른쪽 배치
		none

	  float 속성 해제 : clear 속성
		: float 속성을 요소에 사용하면 다음에 넣는 다른 요소들에도 똑같은 속성이 전달되어,
		  float 속성이 더이상 유용하지 않다고 알려주는 속성이 필요한데 그것이 clear 속성이다.

		left	: float : left 적용했을경우 해제
		right	: float : rigth 적용했을 경우 해제
		both	: 좌우 상관없이 무조건 기본상태로 되돌릴때 사용
