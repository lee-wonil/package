1. 제어문
	- 조건문	: if, switch
	- 반복문	: while, for, do-while, 업그레이드 for문 
	- 보조제어문	: break, continue
			break :강제종료
			continue : 주로 반복문 안에서 사용되면 다음 실행문을 건너 뛰고 시작점(조건식 검사)으로 돌아감

2. 반복문 while

	: 조건식을 먼저 검사한 후, 실행하는 반복문(루프)
	  실행문의 영역{}안에서 루프를 종료할 수 있게 만들어 줘야한다.

	1)구조
		#1. 반복의 횟수를 알때
		변수 = 초기값;		
		while(조건식){
			//반복해서 실행하고 싶은 명령문들...
			증감식;
		}
		#2. 무한반복 : 반복의 횟수를 알지 못할때 이용.
		
		while(true){
			//실행문들
			if(조건) break; //무한반복을 빠져나가는 종료시점 반드시 필요
		}
		
		


3. 반복문 do-while

	선처리 후비교 : 실행 먼저한 후에 조건을 검사하는 형태 : while 문 위치 바꾼것
	
	1) 구조
		do{
			//반복해서 실행할 문장들 ...;
		}while(조건식);
		
	2) 특징 : 영역내의 코드가 한번은 무조건 실행된다.

4. 반복문 for


	지정된 수 만큼 반복해서 실행
	
	1) 구조
		for(초기식;조건식;증감식){
			//실행문들...
		}

		초기식 : 시작하는 값. 처음 for문 시작시 한번만 실행
		조건식 : if문에 들어가는 조건식, 결과는 true/false, true 일때만 실행
		증감식 : 증감연산자 많이 사용
		세미콜론 : 각각의 식이 끝났음을 알려주는 구분기호.

	2) 중첩반복문 : 반복문안에 반복문

	int [] nums = {1,2,3,4,5};
