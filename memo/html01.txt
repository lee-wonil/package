4. 태그

	태그는 소문자로 사용
	여는 태그와 닫는 태그 정확히
		태그	: 쌍으로 이루어짐 ex)<h1>....</h1>, <html></html>
		홑태그	: 하나의 태그로 이루어짐 ex) <img />,<br />

5. HTML 문서 구조
	<!doctype html> : 문서 유형을 지정하는 선언문.
	
	<html></html>	: 웹 문서의  시작과 끝을 알려주는 태그.<head>,<body>를 포함
			  <html lang ="ko"> : 문서에서 사용할 언어 속성으로 지정가능.\

	<head></head>	: 브라우저에게 정보를 주는 태그
			  화면에 보이지 않지만 브라우저가 알아야할 정보를 모두 <head> 태그 안에 입력
			  주요 태그 : <title>, <meta>,<style>,<link>......
	<title></title>	: 웹브라우저 제목 표시줄 지정

	<meta / > 	: 문자 인코딩 및 문서 키워드, 요약 정보

	<body></body>	: 실제 브라우저에 표시될 내용

6. 텍스트 관련 태그
	
	<h1></h1>	: h 태그 : 제목표시할때,<h1>~<h6>
	
	<p></p>		: 단락 만들기 : 앞뒤 줄바꿈 있는 텍스트 덩어리

	<br/>		: 줄 바꿈 기능 : 홑태그

	<hr/>		: 수평 줄 넣기 : 홑태그

	<blockquote></blockquote> : 인용문 넣기 : 양끝 들여쓰기 적용됨

	<pre></pre> 	: 입력하는 그대로 화면에 표시 : HTML에서는 공백은 한개만 적용되어, <pre>태그 사용시 소스 그래도 표시됨

	<span></span>	: 줄바꿈 없이 영역 묶기 : 일부 텍스트만 묶어서 스타일 적용할 때 주로 사용

	<strong></strong>: 볼드체

	<u></u>		: 밑줄

7. 목록 태그
	<ul>,<li>	: 순서 없는 목록 만들기	: unorder list 태그 안에 li(list item) 태그를 사용해 각 항목을 표시
	<ol>,<li>	: 순서 목록 만들기 : 	order list
			1) type 속성
			2) start 속성 : 중간 번호 부터 시작가능
			3) reversed 속성: 항목을 역순으로 표시

8. 표만드는 태그

	표 : table은 행(row)과 열(column)으로 이루어져 있다.
	
	<table></table>	: 테이블 영역에 해당
	<tr></tr>	: 행을 지정
	<td></td>	: 셀 만들기
	<th></th>	: 제목 셀 만들기

	- 표만들기
	<table> 태그 안에 행의 수 만큼 <tr> 태그를 넣고, 
	각<tr>태그 안에 열의 수만큼 셀의 한칸한칸에 해당하는 <td> 태그를 만든다.

	- 행/열 합치기
	: 셀을 합치는 것으므로 셀을 만드는 <td>태그에서 이루어짐

	colspan 속성 : 열합치기
	rowspan 속성 : 행합치기

	<caption></caption>		: 표 제목 붙이기 ( 중앙 정렬 )
	<figcaption></figcaption>	: 표 제목 붙이기 ( 중앙 정렬 X)

9. 이미지 태그 

	- 이미지 파일
		GIF : 색상수 256가지뿐, 파일 크기 작음. 아이콘, 불릿등 작은 이미지에 주로 사용. 
		      움직이는 이미지 만들수 있는 장점
		JPG : 사진. GIF보다 다양한 색상과 명암 표현 
		PNG : 투명 배경. 다양한 색상 표현.

	- 태그
		<img /> : 홑태그, src 속성을 사용하여 이미지 파일 경로 지정해줘야함
			src = "이미지 경로"
			alt = "이미지 설명글"
	
	- src 속성 : 이미지 경로 : 반드시 지정해줘야 하는 속성

		# 내컴퓨터의 이미지 파일 경로
		  현재 HTML 문서에서 이미지 파일까지 찾아가기 위한 경로
		
		상대경로 : 현재 작업파일을 기준으로 이미지 파일의 위치를 작성 (확장자명 포함)
			   작업 파일과 동일한 폴더내의 이미지가 있을 경우, 이미지 파일 이름만 작성
			   작업 파일 내에 있는 하위 폴더 안에 이미지가 있을 경우,
			   한 단계 아래를 뜻하는 '/'를 사용하여 폴더명/파일명으로 작성
			   ex) img/이미지명.jpg
			   작업 파일의 상위 폴더에 있을 경우 '..'를 사용하여, '.. 이미지명' 작성해주기.
			   ex) ../ref/이미지명.jpg
	
		절대경로 : 파일이 컴퓨터 내에 위치한 전체 주소 작성
			   ex) D:\workspace\0909

		웹 상의 링크를 이용한 이미지 경로
			: 웹브라우저에서 이미지를 선택하여 마우스 우클릭으로 이미지주소 복사
	
	- alt 속성 : 이미지 설명글 : 옵션
		
		시각장애인들이 사용하는 화면 낭독기는 문서상의 텍스트만 읽어주기 때문에
		alt를 이용하여 설명을 넣어주면 화면 낭독기가 내용을 읽어줌
		
	- width, height 속성 : 이미지 가로 세로 사이즈 지정
	
	- 이미지 설명글
		<figure><figcaption>

10. 링크
	- 태그
		<a></a>

		<a href="링크할 주소">텍스트...</a>: 	텍스트 클릭시 이동

			  현재창 : _self(기본값)
		download : 링크한 내용을 보여주는 것이 아니라 다운로드 함
		hreflang : 링크한 문서의 언어 지정

	- 앵커
		<태그 id="앵커명"> 텍스트 or 이미지 </태그>
		<a href = "#앵커명> 텍스트 or 이미지</a>


11. 폼 관련 태그(*)

	아이디와 비번 입력, 로그인 버튼, 회원가입창 등 사용자가 웹사이트로 정보를 보낼 수 있는
	요소들을 모두 form(폼)이라 한다.
	
	- form 태그
		<form 속성="속성값"> 여러가지 form 요소</form>

	- form 태그 속성
		method	: 사용자가 입력한 내용들을 서버쪽으로 어떻게 넘겨줄지 지정
			[속성값]
			get : 주소 표시줄에 사용자가 입력한 내용이 그대로 드러남. (기본값) 
			post : 많이 사용 방법. 입력내용이 드러나지 않고 길이 제한 받지 않음.

		name	: 폼태그/요소태그의 이름. 한 문서안에 여러개의 form 태그/요소태그가 있을 경우 구분하기 위해 사용	

		id	: 태그에 id 값을 지정. 한 문서에서 유일한 id값을 줄때

		action 	: form 태그 안의 내용들을 처리해줄 서버상의 페이지 경로 지정

		target 	: action 태그에서 지정한 페이지를 현재 창이 아닌 다른 위치에 열도록 지정.
	
	- class와 id
		class=""	: 이름을 주다. 같은 특성을 가진 아이들을 grouping 한다
				  띄어쓰기 구분
		id = ""		: 클래스 보다 우선순위가 높은 선택자. 단 한 태그의 하나의 속성값을 갖는다. 	

12. input 태그
	
	홑 태그, 폼에서 사용자가 입력하는 부분을 만들때 사용
	한줄 글자, 체크박스, 로그인버튼 등 여러타입이 존재

	- input 태그의 type 속성의 속성값 <input type=""속성값.../>

	text		: 한줄짜리 텍스트 입력 할 수 있는 텍스트 상자
	hidden		: 화면상에 보이지 않지만 서버로 전송할때 함께 전송되는 요소 (관리자만 알아야할 것 입력)
	password	: 비밀번호 입력란. 내용이 * 나 점으로 표시
	radio		: 하나의 항목만 선택할 수 있는 기능. name 속성값이 동일해야 동일한 그룹으로 인식
	checkbox	: 두개 이상의 여러가지 선택가능
	color		: 색상 선택 상자
	date, month,week: 날짜(년,월,일), 월 년,주 년 선택: 브라우저별 지원가능 여부 체크
	time, datetime-local: 시간 날짜와 시간 지정
	reset		: 입력된 모든 정보 지우기.
	submit		: 사용자가 폼에 입력한 정보를 서버로 전송. value 속성값이 버튼의 표시글이 됨.
	image		: 이미지 버튼 넣기. submit 기능과 같이 입력정보 서버로 전달. src = "이미지 경로"
	file		: 파일 첨부. 첨부시 form의 method 속성 post (추후)

	- input 태그의 다른 속성들.....

	autofocus	: 입력 커서 표시하기
	placeholder	: 입력 박스에 기입할 내용 힌트 표시하기
	readonly	: 읽기 전용 필드
	required	: 필수 기입 필드 지정

13. selector (*)
	: 옵션중에 선택하는 드롭다운 목록
	
	- 태그
		<select>
			<option value="값1"> 내용1 </option>
			<option value="값2"> 내용2 </option>	
			<option value="값3"> 내용3 </option>	
		</select>

	- option 속성
		value 		: 옵션을 선택했을때 서버로 넘겨질 값 지정
		selected	: 화면에 표시될 때 기본적으로 선택되어 있는 옵션을 지정

	- optgroup 태그
		: 항목들을 그룹으로 묶어야 할 경우 사용


14. textarea (*)
	: 여러줄 입력하는 텍스트 영역
	
	- 태그
		<textarea> 내용 .....</textarea>

	- 속성
		cols : 텍스트 영역의 가로 너비를 문자 단위로 지정
		rows : 텍스트 영역의 세로 너비를 줄단위로 지정

15. button (*)
	: 버튼 태그. <input type="button"...>과 동일한 버튼으로 사용 가능
	  차이점은 화면 낭독기에서 버튼이 있다는 것을 정확히 인지하고 전달.
	  css 활용에 편리
	
	- 태그
		<button>버튼에 적힐 문자</button>
	- type 속성값
		submit : 폼을 서버로 전송
		reset  : 입력내용 초기화
		button : 버튼 형태만 만들고 자체 기능 X

16. progress
	: 작업의 진행상태를 나타내는 등의 막대 그래프 값은 태그
	
	- 태그
		<progress value ="값">....</progress>
	
	- 속성
		value : 부동소수점으로 표현 0보다 크거나 같고 max 값보다 작거나 같아야. max 미지정시 1.0보다 작아야함
		max   : 완료 기준으로 부동소수점으로 표현 1.0보다 작아야함
